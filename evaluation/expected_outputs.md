# Expected Outputs for Evaluation Scenarios

## Scenario 1 — High Risk Rough Sleeping in London
- A1 should classify as **High Risk**.
- A2 should return services such as:
  - St Mungo’s Emergency Shelter
  - Glass Door Winter Night Shelter
  - Outreach teams
- A3 should list ID documents + GP letter.
- A4 should generate an HTML plan (PDF optional).

## Scenario 2 — Domestic Abuse Refuge Case
- A1 should detect **domestic abuse priority need**.
- A2 should recommend **Women's Aid Refuge** and advice services.
- A3 should emphasise refuge documentation needs.
- A4 should output an HTML plan referencing DA safeguarding.

## Scenario 3 — Family with Children Facing Eviction
- A1 should classify as **priority need (children)**.
- A2 should match **Manchester Homelessness Hub**.
- A3 should recognise prevention duty context.
- A4 should create a structured family plan.
